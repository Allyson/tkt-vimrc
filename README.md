# Vimrc

This is a vim installation bundle based on Vim Bootstrap
with some personal customizations.

## Installation

To install, just follow this steps

    cp ./generate.vim ~/.vimrc
    cp ./vimrc.local ~/
    vim +NeoBundleInstall +qall
